## Dependencies

- Python 3.12
  - Recommended:
    [pyenv](https://github.com/pyenv/pyenv?tab=readme-ov-file#installation)
  - `pyenv install 3.12`
  - `pyenv local 3.12`
- Poetry (https://python-poetry.org/)
- `pip install pyright` (pre-commit hook uses local pyright for linting)

cd blog
